# reactjs-demo-7plus
这个是新开的学习reactJS的工程，专注学习reactjs基础。并学会在微信公众号页面使用微信支付，支付宝支付。

# 如何运行

安装

```npm intall ```

运行开发模式

```npm run dev ```

生产环境打包

```npm run dev ```

打开http://localhost:7777进行预览
